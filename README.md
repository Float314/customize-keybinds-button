# Customize Keybinds Button

A small Geometry Dash Geode mod that adds a "Customize Keybinds" button to the pause menu in both the Editor (soon) and Play modes.

The button opens the keybind customization menu directly, making it easier to adjust your controls while playing or testing levels.

Tbh, this mod is kinda useless for most players. But for people like me who wants to constantly adjust their keybinds just for the sake of comfort, this mod is useful ig. 

This is only my second Geode mod ever, so don’t expect anything revolutionary. I mainly made it because I personally needed it, and figured someone else might find it helpful too.

Idk If this mod will end up in Geode's Index but who knows???
